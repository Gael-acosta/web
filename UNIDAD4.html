Ejemplo de cómo crear un sitio web en internet

Introducción

Crear un sitio web en internet puede parecer una tarea abrumadora, pero con los pasos correctos, puede ser un proceso bastante directo. Este ejemplo te guiará a través de la creación de un sitio web simple utilizando HTML, CSS y un servicio de hosting gratuito.

Paso 1: Planificación del sitio web

Antes de comenzar a codificar, es importante planificar tu sitio web:

  • Define el propósito de tu sitio web
  • Esboza la estructura y el contenido de las páginas
  • Decide sobre el diseño básico

Paso 2: Crear el HTML básico

Comienza creando un archivo llamado index.html con el siguiente contenido:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primer Sitio Web</title>
    <link rel="stylesheet" href="estilos.css">
</head>
<body>
    <header>
        <h1>Bienvenido a Mi Sitio Web</h1>
    </header>
    <nav>
        <ul>
            <li><a href="#">Inicio</a></li>
            <li><a href="#">Acerca de</a></li>
            <li><a href="#">Contacto</a></li>
        </ul>
    </nav>
    <main>
        <h2>Contenido Principal</h2>
        <p>Este es un ejemplo de párrafo en mi sitio web.</p>
    </main>
    <footer>
        <p>&copy; 2024 Mi Sitio Web</p>
    </footer>
</body>
</html>

Paso 3: Añadir estilos con CSS

Crea un archivo llamado estilos.css en el mismo directorio y añade los siguientes estilos:

body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    margin: 0;
    padding: 0;
}

header, footer {
    background-color: #333;
    color: #fff;
    text-align: center;
    padding: 1rem;
}

nav ul {
    list-style-type: none;
    padding: 0;
}

nav ul li {
    display: inline;
    margin-right: 10px;
}

nav ul li a {
    color: #333;
    text-decoration: none;
}

main {
    padding: 20px;
    max-width: 800px;
    margin: 0 auto;
}

Paso 4: Elegir un servicio de hosting

Para este ejemplo, usaremos GitHub Pages, un servicio de hosting gratuito:

  1. Crea una cuenta en GitHub si aún no tienes una
  2. Crea un nuevo repositorio llamado username.github.io, donde "username" es tu nombre de usuario de GitHub
  3. Sube tus archivos HTML y CSS a este repositorio

Paso 5: Publicar tu sitio web

Una vez que hayas subido tus archivos a GitHub:

  1. Ve a la pestaña "Settings" de tu repositorio
  2. Desplázate hacia abajo hasta la sección "GitHub Pages"
  3. En "Source", selecciona la rama "main"
  4. Tu sitio web ahora estará disponible en https://username.github.io

Conclusión

¡Felicidades! Has creado y publicado tu primer sitio web. Recuerda que este es solo un ejemplo básico. A medida que te familiarices más con el desarrollo web, podrás añadir más funcionalidades, mejorar el diseño y crear sitios web más complejos.
